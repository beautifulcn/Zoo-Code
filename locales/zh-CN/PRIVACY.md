# Zoo Code 隐私政策

**最后更新日期：2026年5月13日**

Zoo Code 尊重您的隐私，并致力于透明地处理您的数据。以下是关于关键数据去向（以及更重要的，它们不去向哪里）的简要说明。

## **您的数据去向（以及不去向哪里）**

- **代码与文件**：Zoo Code 在需要提供 AI 辅助功能时会访问您本地机器上的文件。当您向 Zoo Code 发送命令时，相关文件可能会被传输到您选择的 AI 模型提供商（例如 OpenAI、Anthropic、OpenRouter）以生成响应。如果您选择 Zoo Code Cloud 作为模型提供商（代理模式），您的代码可能会经过 Zoo Code 服务器，但仅用于转发至上游提供商。我们不会存储您的代码；代码在转发后立即删除。否则，您的代码将直接发送给提供商。AI 提供商可能会根据其隐私政策存储数据。
- **命令**：通过 Zoo Code 执行的任何命令都发生在您的本地环境中。但是，当您使用 AI 驱动的功能时，命令中的相关代码和上下文可能会被传输到您选择的 AI 模型提供商（例如 OpenAI、Anthropic、OpenRouter）以生成响应。我们无法访问或存储此数据，但 AI 提供商可能会根据其隐私政策进行处理。
- **提示词与 AI 请求**：当您使用 AI 驱动的功能时，您的提示词和相关的项目上下文会发送到您选择的 AI 模型提供商（例如 OpenAI、Anthropic、OpenRouter）以生成响应。我们不存储或处理这些数据。这些 AI 提供商有自己的隐私政策，并可能根据其服务条款存储数据。如果您选择 Zoo Code Cloud 作为提供商（代理模式），提示词可能会经过 Zoo Code 服务器，但仅用于将其转发给上游模型，且不予存储。
- **API 密钥与凭据**：如果您输入 API 密钥（例如，用于连接 AI 模型），它会存储在您设备的本地，绝不会发送给我们或任何第三方，除非是您选择的提供商。
- **遥测（使用数据）**：我们收集功能使用情况和错误数据，以帮助我们改进 Zoo Code。此遥测由 PostHog 提供支持，包括您的 VS Code 机器 ID、功能使用模式和异常报告。VS Code 机器 ID 是一个持久性标识符，在某些司法管辖区可能被视为个人数据；我们仅将其用于产品分析和错误分组。我们仅在产品分析和调试需要的时间内保留遥测数据。遥测**不会**收集您的代码或 AI 提示词，您可以随时通过设置退出。
- **Zoo Code 可观测性（仅限已认证的订阅用户）**：如果您登录 Zoo Code 并拥有有效订阅，Zoo Code 将向 Zoo Code 后端 (zoocode.dev) 发送 LLM 使用遥测数据。这包括任务 ID、AI 提供商名称、模型名称、令牌计数（输入/输出/缓存）和预估成本。此数据与您的已认证 Zoo Code 帐户关联。您可以随时通过聊天区域的 Zoo Code 徽章退出登录，从而停止此类数据收集。
- **市场请求**：当您在市场（Marketplace）中浏览或搜索模型配置配置文件 (MCP) 或自定义模式时，Zoo Code 会向 Zoo Code 的后端服务器发出安全的 API 调用以获取列表信息。这些请求仅发送履行请求所需的查询参数（例如扩展版本、 search 词），不包括您的代码、提示词或个人可识别信息。

## **我们如何使用您的数据（如果已收集）**

- 我们使用遥测数据来了解功能使用情况并改进 Zoo Code。
- 我们**不**出售或分享您的数据。
- 我们**不**使用您的数据训练任何模型。

## **您的选择与控制**

- 您可以本地运行模型，以防止数据被发送到第三方。
- 默认情况下启用遥测收集以帮助我们改进 Zoo Code，但您可以随时通过设置退出。
- 您可以删除 Zoo Code 以停止所有数据收集。

## **安全与更新**

我们采取合理措施保护您的数据，但没有任何系统是 100% 安全的。如果我们的隐私政策发生变化，我们将在扩展程序内通知您。

## **联系我们**

对于任何与隐私相关的问题，请通过 support@zoocode.dev 与我们联系。

使用 Zoo Code 即表示您同意本隐私政策。
