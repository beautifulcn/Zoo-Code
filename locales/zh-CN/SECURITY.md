# 安全政策

## 支持的版本

我们仅积极为 Zoo Code 的最新次要版本（minor release）提供补丁。旧版本是否获得修复由我们酌情决定。

## 报告漏洞

请发送邮件至 support@zoocode.dev，并包含以下内容：

- 问题的简要总结
- 重现步骤或概念验证（PoC）
- 任何可能帮助我们理解问题的日志、堆栈跟踪或截图

我们会在 48 小时内确认收到报告，并争取在 30 天内发布修复或缓解措施。在我们努力解决问题期间，请对相关细节进行保密。

感谢您帮助我们确保 Zoo Code 用户的安全。
